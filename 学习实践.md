

## 服务器配置需求
ubuntu 环境，是最方便的

## 安装redis
ubuntu 安装 redis；

```

$sudo apt-get install redis-server

```

安装后操作系统直接帮启动  redis 服务；都不需要自己过多操心；

```
$ ps -ef |grep redis
redis    15886     1  0 18:36 ?        00:00:00 /usr/bin/redis-server 127.0.0.1:6379
```

## 安装mysql
安装mysql 跟redis 也是类似的；非常方便
